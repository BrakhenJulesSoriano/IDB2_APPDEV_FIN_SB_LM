CSS Basics

Element Selector:

This example selects all h1 elements and applies a blue color and a font size of 24 pixels.

Class Selector:

This example selects all elements with class="highlight" and applies a yellow background color and bold font weight.

ID Selector:

This example selects the element with id="header" and applies a gray background color and a height of 100 pixels.

These examples illustrate some fundamental concepts of CSS. By using different selectors, properties, and values, you can customize the appearance of your web pages in various ways.
