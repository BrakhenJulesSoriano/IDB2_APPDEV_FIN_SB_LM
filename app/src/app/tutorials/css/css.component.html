CSS Introduction

CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation and appearance of a document written in HTML (Hypertext Markup Language). It provides a way to control the layout, formatting, colors, and other visual aspects of a web page, allowing web developers to separate the content of a page from its presentation.

CSS works by selecting HTML elements and applying styles to them. It uses a set of rules to define how elements should be styled. These rules consist of selectors and declarations. The selector specifies which elements the rule applies to, and the declaration contains one or more property-value pairs that define the specific styles to be applied.
