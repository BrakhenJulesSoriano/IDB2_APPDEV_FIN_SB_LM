Selectors and declarations

In CSS, selectors are used to target specific HTML elements, while declarations define the styles to be applied to those elements. Selectors can be based on the element name, class, or ID, as well as various attributes. For example, the selector "h1" targets all 'h1' elements in the HTML document, while ".my-class" targets all elements with a class of "my-class". Similarly, "#my-id" targets an element with the ID "my-id". Once a selector is defined, declarations are used to specify the styles to be applied to the targeted elements. Declarations consist of a property and a value separated by a colon, and are enclosed in curly braces as a block. For example, the declaration "color: red;" sets the text color of the selected element to red, while "font-size: 24px;" sets the font size to 24 pixels example:


Selectors and declarations can also be combined to target specific elements and apply styles to them. For example, the selector "h1.my-class" targets all "h1" elements with a class of "my-class", while the declaration block enclosed in curly braces sets the font size to 24 pixels and the text color to red. Overall, selectors and declarations are essential to CSS as they allow designers to create specific styles for different HTML elements, making the appearance of the website more visually appealing and enhancing the user experience.


Applying styles to HTML elements


Applying styles to HTML elements is a fundamental concept in CSS. CSS allows designers and developers to change the appearance of HTML elements by applying styles to them. This is done by targeting specific elements using CSS selectors and defining the styles to be applied using CSS declarations. CSS declarations can also include multiple properties and values, allowing for more complex styling options. Additionally, CSS allows for the use of inheritance and cascading, which can further simplify and streamline the styling process.


Examples

In addition to the basic CSS styling examples given, there are many other properties and values that can be used to create more complex and dynamic styles. For example, the "border" property can be used to add borders around HTML elements, with various styles and thicknesses. The "text-align" property can be used to align text within an element, such as centering or justifying it. The "padding" and "margin" properties can be used to adjust the spacing around elements, either inside or outside of the element's borders. In addition to these individual properties, CSS also supports shorthand properties that can be used to define multiple styles at once. For example, the "font" shorthand property can be used to define the font family, size, weight, style, and line height all at once. Similarly, the "background" shorthand property can be used to define the background color, image, repeat behavior, and position all at once.
