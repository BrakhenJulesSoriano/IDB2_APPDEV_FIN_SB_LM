In this example, we have a basic HTML table structure with three columns: "Product," "Price," and "Quantity." Here's an explanation of the CSS used:

  1. .my-table: This class is applied to the table element to target it with CSS.
  2. border-collapse: This property is set to collapse to remove the spacing between table cells, creating a seamless appearance.
  3. .my-table th, .my-table td: This selector targets both the th (table header) and (table data) elements within the table.
  4. border: A 1px solid border with the color #ddd is applied to both table headers and table data cells.
  5. padding: A padding of 8px is added to create some spacing between the content and the cell borders.
  6. text-align: The content within table headers and table data cells is aligned to the left.
  7. .my-table th: This selector targets only the table header cells.
  8. background-color: A light gray color (#f2f2f2) is applied as the background color to table header cells.

You can customize the table styles further by modifying the CSS properties and values according to your design requirements.
